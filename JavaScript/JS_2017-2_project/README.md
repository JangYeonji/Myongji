# Little Prince Diary

### Description
인터넷프로그래밍II 과목에서 배운 내용으로 진행한 프로젝트입니다. 이 웹은 현재를 기준으로 나타나는 달력, 날씨와 기분을 기록할 수 있는 일기장, To Do List 작성할 수 있는 다이어리 홈페이지입니다.
